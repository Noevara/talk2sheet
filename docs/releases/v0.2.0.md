# Talk2Sheet v0.2.0

Talk2Sheet `v0.2.0` is a single-sheet analysis depth release.

This version does not expand to cross-sheet joins. Instead, it improves how many business questions can be handled inside one sheet, how stable intent routing is, and how clearly results are explained.

## Highlights

- Period comparison support for:
  - month-over-month
  - year-over-year
  - delta
  - ratio
- Stronger combined planning for filter + groupby + Top N questions
- Better trend planning with day/week/month grain, recent-N windows, and relative month filters
- Structured answer cards with clear conclusion, evidence, and risk note sections
- Chart recommendation and chart metadata alignment across planner, runtime, and frontend
- Chart-mode downgrade visibility with explicit fallback reason
- Result-card follow-up suggestions that prefill the next question in composer
- Intent regression corpus + offline evaluator + CI integration with failure snapshot artifacts

## Included In This Release

### Analysis depth improvements

- Added direct `period_compare` planning branch for relative-period compare questions
- Added question-value filter extraction and merge into selection plans for direct-language filters
- Added recent-period parsing (`last/recent N`) and richer time-grain handling in trend plans
- Added more stable planner metadata for compare, trend, ranking, and share scenarios

### Answer and chart quality improvements

- Added richer rule-based summaries for ranking/share/trend/compare/detail
- Added stricter segment constraints for LLM answers to align with structured output
- Added chart context fields (`title`, axis labels, units, point count, fallback reason)
- Added frontend chart context display and downgrade messaging

### Frontend conversation flow improvements

- Added follow-up suggestion chips on assistant result cards
- Clicking a suggestion now writes text back into the composer draft
- Composer now auto-focuses after suggestion apply, keeping the text editable before send

### Quality governance and CI layering

- Added `v0.2.0` intent regression corpus across:
  - compare
  - filter
  - time aggregation
  - Top N
  - detail/summary
- Added offline evaluator script: `python apps/api/scripts/eval_intent_cases.py`
- Added independent CI job for intent regression
- Added failure snapshot output and artifact upload for easier regression debugging

## Scope Reminder

Supported in `v0.2.0`:

- Workbook-aware single-sheet routing and analysis
- Clarification and follow-up conversation
- Compare / ranking / share / detail rows / trend / basic chart / lightweight forecast
- Structured pipeline visibility and request-level diagnostics

Not included in `v0.2.0`:

- Cross-sheet joins or combined multi-sheet analysis
- Advanced statistics
- Causal inference
- Production-ready object storage
- Persistent session backends

## Validation

Release validation was run with:

- `make ci-check`
- `python apps/api/scripts/eval_intent_cases.py`
