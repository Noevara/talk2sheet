# Talk2Sheet v0.1.0

First public open-source release of Talk2Sheet.

Talk2Sheet lets users ask natural-language questions about Excel and CSV data, routes the question to the right sheet inside a workbook, runs the analysis with pandas, and returns both the answer and the visible execution pipeline.

## Highlights

- Workbook-aware analysis on one sheet at a time
- File upload, sheet listing, and preview
- Natural-language analysis for totals, averages, distinct count, Top N, detail rows, trends, and lightweight forecasting
- Multi-turn conversation with clarification and follow-up context reuse
- User-visible routing summary, execution scope, result tables, and charts
- `auto / text / chart` response mode
- Semantic intent understanding and structured analysis pipeline
- Multilingual UI and documentation in English, Chinese, and Japanese

## Included In This Release

### Product capabilities

- Upload Excel and CSV files from the web UI
- Preview workbook sheets before asking questions
- Route a question to one target sheet inside a workbook
- Clarify the target sheet when multiple candidates are plausible
- Keep context across follow-up questions
- Show structured answers together with routing and execution metadata

### Engineering foundations

- FastAPI backend with pandas-based execution
- Vue 3 + Vite + TypeScript frontend
- File storage abstraction behind `services/storage/`
- OpenAPI-generated contract artifacts and frontend API types
- Layered CI checks for contracts, backend, and frontend
- Baseline request observability with `X-Request-ID` and pipeline timing metadata

## Current Scope

Supported in `v0.1.0`:

- Workbook-aware single-sheet analysis
- Row count, totals, averages, distinct count
- Top N and ranking
- Detail rows
- Trend analysis and basic charts
- Lightweight time-series forecasting
- Clarification and follow-up conversation

Not included in `v0.1.0`:

- Cross-sheet joins
- Combined multi-sheet analysis
- Advanced statistics
- Causal inference
- Production-ready object storage
- Persistent session backends
