# Talk2Sheet v0.3.3

Talk2Sheet `v0.3.3` is the controlled cross-sheet Join Beta release.

This release introduces a constrained join execution path with strict scope boundaries, preflight checks, quality visibility, and automatic fallback to sequential sheet analysis when join quality is risky.

## Highlights

- Added Join Beta scope and guardrails for cross-sheet requests
- Added Join Preflight checks before execution:
  - join key existence
  - key type compatibility
  - null/duplicate risk checks
  - estimated match-rate checks
  - repair suggestions for unsupported/risky cases
- Added controlled Join Beta execution path:
  - two-sheet only
  - single-key equality join
  - `inner` / `left`
  - aggregate-oriented questions (`sum` / `count` / `avg` / `top` / `trend`)
- Added join quality panel and fallback closure:
  - match/unmatched visibility
  - row-expansion risk signal
  - fallback reason when quality thresholds fail
- Updated frontend boundary copy and examples (EN / ZH / JA):
  - supported Join Beta examples
  - out-of-scope join examples
- Added Join Beta feature flag governance:
  - `TALK2SHEET_ENABLE_JOIN_BETA`
  - env-aware default via `TALK2SHEET_APP_ENV` (`dev` on, `prod` off unless explicitly enabled)
- Added independent Join Beta regression checks in CI:
  - `apps/api/tests/fixtures/join_beta_cases.v0.3.3.json`
  - `python apps/api/scripts/eval_join_beta_cases.py`
  - CI job with failure snapshot artifact upload

## Included In This Release

### Join Beta runtime path

- Added join intent signals and controlled routing boundary handling
- Added Join Preflight output in pipeline metadata with actionable suggestions
- Added join executor beta path and join quality evaluation
- Added automatic fallback to sequential decomposition when join quality is below thresholds

### UX and boundary governance

- Updated capability/out-of-scope/routing boundary wording in EN/ZH/JA
- Updated guided examples to separate:
  - supported Join Beta prompts
  - unsupported boundary prompts
- Aligned backend clarification wording with frontend boundary messaging

### Quality and CI

- Added Join Beta evaluator script and fixture corpus
- Added CI layered check and failure snapshot upload for join regressions
- Added config-level join beta gating tests and disabled-mode integration check

## Scope Reminder

Supported in `v0.3.3`:

- workbook-aware routing and sequential multi-sheet flow
- controlled Join Beta execution under strict constraints
- join preflight + quality visibility + automatic fallback
- trend/ranking/compare/detail/basic chart/lightweight forecast (existing single-sheet path)

Not included in `v0.3.3`:

- arbitrary joins/unions
- multi-key joins, multi-hop joins, joins across 3+ sheets
- advanced statistics and causal inference
- production-ready object storage and persistent session backends
