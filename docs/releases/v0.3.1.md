# Talk2Sheet v0.3.1

Talk2Sheet `v0.3.1` is a task-workflow completion release for sequential multi-sheet analysis.

This release turns “A then B” follow-up conversations into a visible step-by-step workflow, improves scope stability across turns, and adds step-level observability for troubleshooting.

## Highlights

- Task-step workflow for sequential multi-sheet analysis:
  - task steps card with status (`pending` / `current` / `completed` / `failed`)
  - one-click `continue_next_step` action
- Analysis anchor carry-over for follow-up scope stability:
  - preserves metric / dimension / time scope in typical follow-up turns
- Previous-vs-current step comparison card (A vs B, non-join):
  - shows key findings side-by-side across sequential sheet steps
- Step-level observability with request linkage:
  - `task_step_started`
  - `task_step_completed`
  - `task_step_failed`
  - fields include `request_id + step_id + sheet`
- Synchronized user-facing documentation in English / Chinese / Japanese for the v0.3.1 scope

## Included In This Release

### Task workflow and follow-up UX

- Added task-step state payloads across pipeline and SSE metadata
- Added continue-next-step follow-up action routing (`followup_action=continue_next_step`)
- Added UI rendering for task steps and continue button

### Scope stability and comparison

- Added analysis anchor generation, persistence, and reuse in follow-up context
- Added fallback behavior to avoid overriding explicit user intent when parsing follow-ups
- Added previous-vs-current step comparison payload and frontend comparison card

### Observability and reliability

- Added standardized task-step event logging helper in observability layer
- Added transition detection for started/completed events
- Added failure-path task-step event logging (`task_step_failed`) with error type
- Expanded backend/frontend regression tests for task workflow and comparison behavior

## Scope Reminder

Supported in `v0.3.1`:

- Workbook-aware routing with one-sheet execution per turn
- Sequential multi-sheet analysis workflow (A then B) with visible task steps
- Continue-next-step action and follow-up scope carry-over
- Ranking / trend / compare / detail / chart / lightweight forecast
- Request-level and step-level observability for troubleshooting

Not included in `v0.3.1`:

- Cross-sheet joins in one step
- Unrestricted combined multi-sheet execution
- Advanced statistics and causal inference
- Production-ready object storage and persistent session backends
